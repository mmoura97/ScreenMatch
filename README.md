# 🎬 ScreenMatch

Projeto desenvolvido durante meus estudos iniciais de Java na formação da Alura.

Este repositório representa minha primeira versão do ScreenMatch, utilizada para praticar os conceitos fundamentais da linguagem antes de avançar para Programação Orientada a Objetos.

## 📚 Conceitos praticados

* Variáveis e tipos primitivos
* Entrada e saída de dados
* Operadores matemáticos
* Estruturas condicionais (`if` / `else`)
* Estruturas de repetição (`for` e `while`)
* Métodos
* Manipulação básica de dados

## 🛠️ Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git
* GitHub
* Linux Ubuntu

## 🚀 Objetivo

Aplicar na prática os fundamentos da linguagem Java através do desenvolvimento de uma aplicação simples inspirada em plataformas de filmes e séries.

## 📂 Estrutura do projeto

```text
src/
```

## ▶️ Como executar

1. Clone o repositório.
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe principal da aplicação.
